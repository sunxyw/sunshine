## Installation

- Install with npm

```bash
  npm install
```

- Install with yarn

```bash
  yarn
```

## Run Locally

Start the server

```bash
  npm run serve
  # or
  yarn serve
```

## Deployment

To deploy this project run

```bash
  npm run build
  # or
  yarn build
```
